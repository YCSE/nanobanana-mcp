# Switch to NanoBanana Flash

Switch to NanoBanana Flash model for faster image generation.

## Action

Use the `set_model` MCP tool from nanobanana-mcp:

```
mcp__nanobanana-mcp__set_model with model="flash"
```

## Confirmation

After switching, confirm: "Switched to NanoBanana Flash (gemini-2.5-flash-image)"
