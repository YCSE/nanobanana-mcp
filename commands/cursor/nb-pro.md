# Switch to NanoBanana Pro

Switch to NanoBanana Pro model for higher quality image generation.

## Action

Use the `set_model` MCP tool from nanobanana-mcp:

```
mcp__nanobanana-mcp__set_model with model="pro"
```

## Confirmation

After switching, confirm: "Switched to NanoBanana Pro (gemini-3-pro-image-preview)"
